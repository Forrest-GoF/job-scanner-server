### 프로젝트 기간
22.07~

### 목적 및 필요성

- 수시 채용 확대로 채용 공고가 증가하면서 채용 공고 검색을 어려워한다.
- 취업준비생 81%, “공채 일정관리를 어려워한다.”
- 취업준비생 59%, “마감일 놓쳐 입사지원 못 한 경험 있다.”

### 프로젝트 개요

- 채용 공고를 모아주고 검색과 필터 기능을 제공한다.
- 채용 진행 사항과 채용 히스토리를 관리할 수 있는 서비스를 제공한다.
